### 기능
- accounts
	- 회원가입
	- 로그인( 기본로그인, 소셜로그인 )
	- 로그아웃
- photos
  - 사진 게시
  - 사진 세부사항
  - 사진 게시물 댓글
  - 사진에 관한 태그
- columns
  - 칼럼 게시

  - 칼럼 댓글
- blog
  - 여행 블로그 소개
  - 지도 위젯
